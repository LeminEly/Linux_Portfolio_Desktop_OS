# 🖥️ Linux Portfolio Desktop

A web-based interactive simulation of my personal Kali Linux desktop setup, inspired by GNOME and Debian-based systems.

This project recreates a realistic Linux desktop experience inside the browser, designed to feel like exploring my actual PC environment. It is based on my Kali Linux setup but can represent any Debian-based system look and feel.

Instead of a traditional portfolio website, this project turns my desktop into an interactive experience where users can explore personal profile information in a fun and immersive way.

---

## 🚀 Features

- GNOME-style Linux desktop interface
- Kali Linux inspired UI/UX
- Draggable and interactive windows
- File explorer simulation
- CV / profile viewer
- Browser-based experience (no installation required)
- Realistic desktop interaction (like using a real PC)

---

## 🎯 Purpose

This project is created as a creative portfolio to showcase my skills in frontend development and UI/UX design by simulating a real Linux desktop environment in the browser.

It provides an immersive way to present my personal profile in a more engaging and interactive format than a traditional portfolio website.

---

## 🛠️ Tech Stack

This project uses modern web technologies to create a highly interactive and realistic Linux desktop experience.

### ⚛️ Frontend Core
- Next.js (React Framework)
- TypeScript

### 🎨 UI / Design System
- Tailwind CSS
- shadcn/ui (modern UI components)
- Lucide Icons / Phosphor Icons

### ✨ Animations & Interactions
- Framer Motion (smooth UI animations)
- GSAP (advanced animations)

### 🪟 Desktop System (Core Logic)
- Zustand (state management for windows & apps)
- react-rnd (drag & resize windows system)
- dnd-kit (advanced drag interactions)

### 💻 Terminal Simulation
- xterm.js (Linux/Kali terminal experience)

### 📂 Data & File System
- IndexedDB (persistent local storage)
- Dexie.js (wrapper for IndexedDB)

### 🚀 Optional Enhancements
- Three.js (3D effects / boot animations)

---

## 📌 Notes

This project is inspired by real Linux environments (Kali Linux / GNOME). It is a visual and interactive simulation, not a full operating system.
